# Escopo - Compartilha Lanche

## Objetivo
Sistema para que participantes de uma turma ou evento informem o que vão levar no próximo encontro, evitando repetições e organizando melhor o lanche coletivo.

## Entidades
- **User**: usuário do sistema (nome, email, senha)
- **SnackContribution**: contribuição de lanche (item, data, id do usuário)

## Funcionalidades
- Cadastro, login e logout
- Criar contribuição de lanche
- Listar contribuições
- Editar própria contribuição
- Excluir própria contribuição

## Escopo Implementado
Todas as funcionalidades acima

## Escopo Não Implementado

- Sistema de avaliação de lanches
