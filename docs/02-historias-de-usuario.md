# Histórias de Usuário

## HU01 - Cadastrar usuário
**Como** visitante,
**Quero** criar uma conta no sistema,
**Para** poder participar da organização dos lanches.

## HU02 - Fazer login
**Como** usuário cadastrado,
**Quero** fazer login no sistema,
**Para** acessar minhas contribuições.

## HU03 - Criar contribuição
**Como** participante logado,
**Quero** informar o lanche que vou levar em uma data,
**Para** que os outros saibam e evitem repetições.

## HU04 - Listar contribuições
**Como** participante,
**Quero** visualizar todas as contribuições cadastradas,
**Para** acompanhar a organização do evento.

## HU05 - Editar contribuição
**Como** participante logado,
**Quero** editar minha contribuição,
**Para** corrigir ou alterar o que vou levar.

## HU06 - Excluir contribuição
**Como** participante logado,
**Quero** excluir minha contribuição,
**Para** cancelar minha participação.
